RespiraWorks Ventilator
README

This directory contains 5s readings of steady-state fan pressure measured at different fan power levels.

For example, the file fan087 contains pressure readings with the blower at 87% power.

I measured as follows.

  • I hacked controller.cpp so that
    • the inspiratory pinch valve stays open,
    • the expiratory pinch valve stayed closed, and
    • fan power is configurable via a DebugVar.
